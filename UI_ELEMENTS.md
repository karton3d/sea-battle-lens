# UI Elements

## Buttons

- **Single Player** — start single player mode
- **Play with Friend** — start multiplayer mode
- **Start** — begin game after setup
- **Play Again** — restart after game over

## Text / Hints

- **Title** — "Fleet Yeet!"
- **Status Text** — current game state messages
- **Hint Text** — player instructions ("Tap opponent's grid", etc.)
- **Result Text** — shot results ("Hit!", "Miss!", "Object destroyed!")
- **Winner Text** — game over message

## Grid Labels

- **Your Grid** — player's grid label
- **Opponent Grid** — opponent's grid label
